Вопросы по курсу

### ВАРИАЦИОННО-ПРОЕКЦИОННЫЕ МЕТОДЫ В МАТЕМАТИЧЕСКОЙ ФИЗИКЕ

обязательный курс для магистров 1-ого года обучения

(2024/2025) осенний семестр

    1. Метода Ритца на примере задачи об упругой пластине.

    2. Энергетический метод для положительно определённых операторов в гильбертовом пространстве. Теорема о функционале энергии. Примеры положительно определенных дифференциальных операторов.

    3. Энергетический метод для только положительных операторов.

    4. Исследование задачи о минимуме функционала энергии. Пространства HА.

    5. Теорема о сходимости минимизирующей последовательности для функционала энергии.

    6. Процесс Ритца построения минимизирующей последовательности энергетического функционала.

    7. Теорема о сходимости метода Ритца.

    8. Построение обобщенного решения краевой задачи Дирихле для уравнения Штурма-Лиувилля. Доказательство положительной определённости оператора краевой задачи.

    9. Основные свойства обобщенного решения задачи Дирихле. Принадлежность решения задачи Дирихле пространству H1 ([a , b]). Обобщённые производные.

    10. Третья краевая задача для уравнения Штурма-Лиувилля. Положительная определённость оператора задачи. Построение минимизирующей последовательности. Естественные краевые условия.

    11. Краевая задача для ОДУ четного порядка. Краевая задача для системы ОДУ.

    12. Построение обобщенного решения для уравнения Пуассона с условиями Дирихле. Неравенство Фридрихса.

    13. Третья краевая задача для уравнения Пуассона. Неравенство Фридрихса-Стеклова.

    14. Задача Неймана для уравнения Пуассона. Неравенство Пуанкаре.

    15. Постановка задачи Дирихле для уравнения Лапласа. Обобщённое решение.

    16. Подходы к решению задачи Дирихле для эллиптического уравнения 2 порядка с неоднородными краевыми условиями.

    17. Степенные сплайны нулевой степени дефекта 1. Теоремы аппроксимации.

    18. Степенные сплайны дефекта 1. Основные теоремы аппроксимации сплайнами первой степени дефекта 1.

    19. Применение сплайн-аппроксимации в методе Ритца для получения приближённых решений.

    20. Билинейные сплайны в плоской области и их применение к построению приближенного решения задачи Дирихле для уравнения Лапласа.

    21. Построение проекционной-сеточной схемы для краевой задачи для ОДУ 2 порядка. Анализ сходимости в L_2и W_2^1.

    22. Построение проекционной-сеточной схемы для задачи Дирихле для эллиптического уравнения 2 порядка. Анализ сходимости в L_2и W_2^1.

    23. Задача на собственные значения для симметрического полуограниченного дифференциального оператора. Основные свойства собственных значений и собственных функций. Теорема о спектре положительно определённого симметрического оператора в гильбертовом пространстве.

    24. Вариационная постановка задачи на собственные значения симметричного положительного операторного уравнения. Приложение к основным задачам математической физики.

    25. Метод Ритца в проблеме вычисления собственных значений задачи Дирихле.
